# Cardio Fitness Linear Regression Project

## 📌 Overview
This project predicts user income based on fitness and lifestyle data.

## 🧠 Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit

## 📊 Features Used
- Age
- Education
- Usage
- Fitness
- Miles

## 🚀 How to Run

### Install dependencies
pip install -r requirements.txt

### Train model
cd src
python train.py

### Run app
cd ../app
streamlit run app.py

## 📈 Output
Predicts income based on user input.
